# SmartBite
Smart oral‑health device • Bite‑force sensing • Embedded systems • Ergonomic design

SmartBite is a compact, ergonomic oral‑health device that measures bite force using embedded sensors, BLE communication, and calibration algorithms. It is designed for research, wellness tracking, and interactive applications.

---

## 🚀 Features
- Bite‑force sensing  
- BLE communication  
- Calibration workflow  
- Ergonomic mouthpiece design  
- Embedded firmware architecture
